## Purpose
Turn weaknesses into concrete, prioritized rewrite actions a candidate can execute.

## Inputs
Dimension scores, keyword gaps, format risks, career-switch flag.

## Process
1. Rank gaps by impact-to-effort. Highest-impact ATS/keyword fixes first.
2. For each action, write a before→after bullet rewrite applying STAR + quantification.
3. For career-switchers, reframe transferable skills via O*NET mapping.
4. Tag each item: effort (S/M/L), impact (Low/Med/High), framework reference.

## Outputs
Ordered roadmap table: action, before→after, effort, impact, rationale.

## Quality Gate
- Every item has effort + impact + a before/after or concrete instruction.
- Top 3 items address the lowest-scoring dimensions.
