## Purpose
Choose ≥2 named, citable frameworks and tune weights per role family/seniority so scoring is fair and relevant.

## Inputs
Role family (e.g., software, sales, design), seniority, industry.

## Process
1. Always include ATS keyword coverage + STAR.
2. Add role-specific lenses: technical roles → skills-match emphasis; leadership roles → scope/impact emphasis; creative roles → portfolio link weighting.
3. Adjust weights (e.g., senior roles weight impact/scope higher than keyword coverage).
4. Cite each framework source from SECOND-KNOWLEDGE-BRAIN.md.

## Outputs
Selected framework set + tuned weight vector + citations.

## Quality Gate
- ≥2 named frameworks, each citable.
- Weights sum to 100% and justified for seniority.
